The Problem (The Junior Way): Every time you call appendChild() on the main screen, the Browser does a Reflow (Recalculates positions) and a Repaint (Re-draws colors). Doing this 1000 times is like calling a painter 1000 times to paint one dot each time.

The Solution (The Senior Way): DocumentFragment is a "Virtual Tray". You put all 1000 dots on the tray in memory (Offline). Then, you put the whole tray on the wall in one single move. The browser only reflows/repaints ONCE!
