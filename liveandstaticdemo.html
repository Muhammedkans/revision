Spoon-Feeding Deep Dive:

1. LIVE list is like a CCTV camera. As soon as you click "Add New Box", the number above changes instantly because JavaScript is still connected to the DOM.

2. STATIC list is like a photo. It was taken when the page loaded. Even if you add 100 boxes, this number will not change until you take a "New Snapshot".
